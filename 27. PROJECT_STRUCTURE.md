

# 27. MVP Kod Klasör Yapısı (Project Structure)

## 27.2. Docker Compose Yapılandırması (`docker/docker-compose.yml`)

Projenin tüm bağımlı servislerinin kiracı izolasyon standartlarına (`14. SECURITY_COMPLIANCE.md`) uygun şekilde birbiriyle izole ve yerel olarak çalışmasını sağlayan orkestrasyon dosyası:

```yaml
version: '3.8'

services:
  # 1. Grafik Veri Tabanı (Neo4j Enterprise / Community)
  gcip-neo4j:
    image: neo4j:5.18.0-community
    container_name: gcip-neo4j
    ports:
      - "7474:7474" # HTTP Konsol Portu
      - "7687:7687" # Bolt Protokolü Portu
    volumes:
      - neo4j_data:/data
      - neo4j_logs:/logs
    environment:
      - NEO4J_AUTH=neo4j/gcipSecurePassword123
      - NEO4J_PLUGINS=["apoc"] # Otonom UUID oluşturma için APOC kütüphanesi
      - NEO4J_dbms_security_procedures_unrestricted=apoc.*
      - NEO4J_dbms_security_procedures_allowlist=apoc.*
    networks:
      - gcip-network

  # 2. Hızlı Oturum ve shared_state Belleği (Redis)
  gcip-redis:
    image: redis:5.0.1-alpine
    container_name: gcip-redis
    command: redis-server --appendonly yes # Hot Recovery için AOF aktif
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data
    networks:
      - gcip-network

  # 3. Yerel Çıkarım Motoru (Local LLM - Llama-3-8B-Instruct)
  gcip-local-llm:
    image: ollama/ollama:latest
    container_name: gcip-local-llm
    ports:
      - "11434:11434"
    volumes:
      - ollama_models:/root/.ollama
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: all
              capabilities: [gpu] # GPU hızlandırması
    networks:
      - gcip-network

  # 4. Backend Uygulama Sunucusu (FastAPI)
  gcip-backend:
    build:
      context: ../
      dockerfile: docker/backend.Dockerfile
    container_name: gcip-backend
    ports:
      - "8000:8000"
    environment:
      - NEO4J_URI=bolt://gcip-neo4j:7687
      - NEO4J_USER=neo4j
      - NEO4J_PASSWORD=gcipSecurePassword123
      - REDIS_HOST=gcip-redis
      - REDIS_PORT=6379
      - LOCAL_LLM_API=http://gcip-local-llm:11434
    depends_on:
      - gcip-neo4j
      - gcip-redis
      - gcip-local-llm
    networks:
      - gcip-network

volumes:
  neo4j_data:
  neo4j_logs:
  redis_data:
  ollama_models:

networks:
  gcip-network:
    driver: bridge
```

## 27.5. Güvenlik ve İzolasyon Middleware Yapısı (Multi-Tenancy)

Her HTTP isteğinde `tenant_id` doğrulaması yapan ve Neo4j bağlantısına bu kimliği otomatik enjekte eden FastAPI middleware kodu:

```python
# backend/app/middlewares/tenant_validator.py
from fastapi import Request, HTTPException, status
from starlette.middleware.base import BaseHTTPMiddleware
import logging

logger = logging.getLogger("gcip_security")

class TenantValidatorMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request: Request, call_next):
        # 1. Onboarding veya Genel API Yolları Dışındaki Tüm İsteklerde tenant_id Sorgulanır
        if request.url.path not in ["/docs", "/openapi.json", "/api/v1/onboarding/register"]:
            tenant_id = request.headers.get("X-Tenant-ID")
            
            if not tenant_id:
                logger.warning(f"🚨 Yetkisiz Erişim Girişimi: X-Tenant-ID başlığı eksik. IP: {request.client.host}")
                raise HTTPException(
                    status_code=status.HTTP_400_BAD_REQUEST,
                    detail="Erişim reddedildi: 'X-Tenant-ID' header parametresi zorunludur."
                )
                
            # 2. İstek bağlamına tenant_id parametresi enjekte edilir
            request.state.tenant_id = tenant_id
            
        response = await call_next(request)
        return response
```
